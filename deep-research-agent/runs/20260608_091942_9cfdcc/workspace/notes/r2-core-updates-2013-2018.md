# CS:GO 核心更新与玩法演进 (2013-2018) — Researcher-2

## 关键发现

### 一、Arms Deal 更新 — 皮肤经济起源（2013年8月14日）
- Arms Deal 更新引入武器皮肤系统，建立基于开箱和社区市场交易的虚拟物品经济
- 首批武器箱: CS:GO Weapon Case 和 eSports 2013 Case
- 同时开放 CS:GO Workshop
- M4A1-S 消音步枪随该更新加入游戏
- 来源: [src_073ea5, src_ecde24, src_7e6639, src_2c86f3]
- 证据强度: strong

### 二、Operation 系列（2013-2017）
共 8 个 Operation:
1. Operation Payback: 2013年4月25日 - 2013年8月31日（首个）
2. Operation Bravo: 2013年9月19日 - 2014年2月
3. Operation Phoenix: 2014年2月20日 - 2014年6月11日
4. Operation Breakout: 2014年7月1日 - 2014年10月2日
5. Operation Vanguard: 2014年11月11日 - 2015年3月
6. Operation Bloodhound: 2015年5月26日 - 2015年10月
7. Operation Wildfire: 2016年2月17日 - 2016年7月
8. Operation Hydra: 2017年5月23日 - 2017年11月13日
- 来源: [src_9486ed, src_66bb95, src_650b87, src_8a0d7c, src_8ae0bc]
- 证据强度: strong

### 三、武器平衡
- **AWP 削弱**（2015年3月31日）：大幅降低开镜移动速度，引发巨大争议
- **R8 Revolver 争议**（2015年12月）：上线时极度超模，数天内紧急削弱
- 来源: [src_e42a84, src_51fccf, src_b78a31, src_7eb595, src_94393f]
- 证据强度: strong (AWP) / medium (R8)

### 四、Hitbox 修复（2015年9月15日）
重新设计角色碰撞盒结构，将 hitbox 从 20 减为 19，修复了颈部穿透判定的长期问题。
- 来源: [src_fe9249, src_d83e54, src_0ccddd]
- 证据强度: strong

### 五、地图重制
- Nuke 重制（2016年2月，与 Wildfire 同步）
- Inferno 重制（2016年10月 beta）
- Dust II 重制（2017年2月移出 Active Duty，10月重制版回归）
- 来源: [src_c6d2b0, src_1fcae9, src_998c37, src_04e955, src_718eed]
- 证据强度: strong

### 六、HRTF 声音系统（2016年12月）
引入基于头部相关传输函数（HRTF）的空间音频处理系统，改善 3D 声音定位。初版存在上下定位错误，次日修复。
- 来源: [src_b9d7d0, src_dcc1fa, src_b46889]
- 证据强度: medium

### 七、Panorama UI（2018年）
2018年6月19日 beta 测试，7月初推送正式服。替换自发布以来的 Scaleform UI，全面翻新主菜单、库存、购买菜单和计分板。
- 来源: [src_964777, src_85ea36, src_b51407]
- 证据强度: strong

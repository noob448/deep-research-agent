# CS:GO Hitbox Fix (2015) & Panorama UI (2018)

## Hitbox Fix (September 2015)
- September 15, 2015: Major hitbox update
- Changed hitbox structure: removed UPPER_NECK hitbox (from 20 to 19 hitboxes)
- Bullets penetrating the neck into the head now count as headshots
- Community discussion on Steam and LinusTechTips forums confirms the date
- Sources: src_fe9249, src_d83e54, src_0ccddd

## Panorama UI (June 2018)
- Beta release: June 19, 2018
- Replaced old Scaleform UI (menus, inventory, buy menu, scoreboard)
- HLTV.org reported live release; skins.cash covered features
- Sources: src_964777, src_85ea36, src_b51407

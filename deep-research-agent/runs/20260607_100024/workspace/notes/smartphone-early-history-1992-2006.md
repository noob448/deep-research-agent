# 智能手机早期发展史 (1992–2006) — 研究笔记

## 1. IBM Simon (1992/1994)
- 原型 "Sweetspot" 由 IBM 工程师 Frank Canova 设计，1992 年 COMDEX 展出
- BellSouth 合作商业化，Motorola 拒绝代工，最终 Mitsubishi 制造
- 1994 年 8 月发布，$899，1.5 英寸厚，510g，1 小时通话
- 功能：触屏（触控笔）、Email、FAX、日历、通讯录、记事本、屏上键盘
- 当时被归类为 PDA + 手机，非"智能手机"（该词后由 Ericsson 创造）
- 来源: src_0a6f13 (Mobile Phone Museum), src_6b3f95 (Textline)

## 2. Nokia Communicator 系列 (1996–)
- Nokia 9000 Communicator (1996): CeBIT 宣布，1996/8/15 上市；AMD Elan i386 24MHz，8MB 内存；翻盖设计（外拨号+内 QWERTY），运行 GEOS OS
- 后续型号：9110 → 9210 (2001，首款 Symbian Communicator) → 9500 → 9300 → E90
- 特点是翻盖+全键盘，定位商务用户
- 来源: src_2825a4, src_5f5a31, src_03d4ca

## 3. Symbian OS 演化
- 根源：Psion EPOC32 (1990s)；1998 年 6 月 Symbian Ltd. 成立 (Psion + Ericsson + Motorola + Nokia)
- Ericsson R380 (2000): 首款商用 "Smartphone" 品牌设备，触屏+键盘翻盖
- Nokia 9210 Communicator (2001): 首款开放 Symbian 手机
- Nokia 7650 (2002): 首款 Nokia Symbian 手机 (S60)，首款内置相机 Nokia 手机
- S60 UI 平台 2001 年 11 月发布，可授权给其他厂商
- Symbian 2006 年占智能手机 OS 市场 ~73%
- 来源: src_cd35d4 (Hotbot), src_96fb0d (AllAboutSymbian), src_31df3b, src_a39032

## 4. BlackBerry / RIM
- RIM 1984 年由 Mike Lazaridis & Douglas Fregin 在加拿大 Waterloo 创立
- 1990s 专注无线数据和 Email 技术
- BlackBerry 850 (1999): 首款设备，双向寻呼机形态，拇指键盘，滚轮导航
- BlackBerry 5810 (2002): 首款增加电话功能的 BlackBerry
- 核心：Push Email + BES (BlackBerry Enterprise Server) 企业级安全
- 2000s 中期成为企业移动 Email 的代名词
- 来源: src_383667, src_ec6673, src_3454e2, src_da777f

## 5. Palm OS / Treo 系列
- Palm Computing 1992 年创立，Palm OS 主导 PDA 市场
- Handspring (1998): Palm 创始人 Jeff Hawkins 等另立门户
- Treo 180 (2002): Palm OS 3.5.2H，单色触屏 160×160，DragonBall 33MHz CPU
- Treo 600 (2003): Handspring/palmOne 品牌切换节点，QWERTY + 触屏
- Treo 650 (2004): Palm OS 5.4，经典机型，广泛流行
- Handspring 与 Palm 硬件部门 2003 年合并为 palmOne
- 来源: src_c9cc23, src_11ca9a, src_44f9ab, src_cd191d

## 6. Windows Mobile / Pocket PC
- 基于 Windows CE 内核
- Pocket PC 2000 → Pocket PC 2002 → Windows Mobile 2003 → WM 5.0 (2005)
- HTC 为核心代工厂：Wallaby, Blue Angel, Magician, Universal 等平台
- Orange SPV 系列 (欧洲运营商定制 HTC)
- Compaq iPAQ 系列为 Pocket PC 标志性 PDA
- O2 XDA 系列为代表性的 Pocket PC Phone
- 触控笔 + 类 Windows 桌面 UI
- 来源: src_a15a3c, src_780dac, src_340b2b, src_3ef01d

## 7. 2006 年市场格局
- 全球智能手机出货 7090 万部
- Nokia：56.4% 份额 (4000 万部，同比 +40%)
- Motorola：8.5%，第二，靠 Linux 机型 MING 在中国成功
- OS 份额：Symbian ~73%，Windows Mobile 次之，BlackBerry/Palm 在后
- 消费者需求趋势：触屏、MP3、Wi-Fi/Bluetooth、快处理器、扩展卡
- 来源: src_96fb0d (ABI Research 数据)

## 8. 核心创新总结 (1992-2006)
- 触控笔 + 电阻式触屏 (IBM Simon → Palm → Windows Mobile)
- QWERTY 物理键盘 (Nokia Communicator, BlackBerry, Treo)
- Push Email (BlackBerry BES)
- 移动互联网萌芽 (WAP → 早期 HTML 浏览器)
- 内置相机 (Nokia 7650, 2002)
- 移动 App 生态萌芽 (Palm OS 第三方软件, Symbian S60 apps)
- 无线连接：Wi-Fi, Bluetooth, IR
- 存储扩展卡槽 (SD, MMC)

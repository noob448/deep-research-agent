# 早期智能机时代 (1992–2006) — 研究笔记

## 1. IBM Simon (1992/1994)
- IBM 工程师 Frank Canova 设计原型，代号 "Sweetspot"，1992 年 COMDEX 展出
- BellSouth 推动商业化，Motorola 拒绝代工后由 Mitsubishi 制造
- 1994 年 8 月上市，$899，约 510g，电池仅 1 小时通话
- 功能：触屏（触控笔）、Email、FAX、日历、通讯录、屏上键盘
- 被归类为 PDA + 蜂窝通信组合，"Smartphone"词后来才由 Ericsson 创造
- 来源：src_0a6f13 (Mobile Phone Museum), src_6b3f95 (Textline)

## 2. Nokia 9000 Communicator 及系列 (1996–)
- Nokia 9000 Communicator 于 1996 年 8 月上市，AMD i386 24MHz CPU + 8MB RAM，GEOS OS
- 系列演进：9000 (1996) → 9110 → 9210 (2001，首款 Symbian) → 9500 → 9300 → E90
- 来源：src_2825a4, src_03d4ca, src_5f5a31

## 3. Symbian OS
- 根源：Psion EPOC32 系统
- Symbian Ltd. 1998 年 6 月成立：Psion + Ericsson + Motorola + Nokia 合资
- Ericsson R380 (2000)：首款以 "Smartphone" 品牌推向市场的设备，Symbian OS 5
- Nokia 7650 (2002)：首款 S60 平台 + 首款内置摄像头 Nokia 手机
- 2006 年巅峰：Symbian 占全球智能手机 OS 市场 ~73%
- 来源：src_cd35d4, src_a39032, src_31df3b, src_96fb0d

## 4. BlackBerry (RIM)
- RIM 1984 年创立于加拿大 Waterloo
- BlackBerry 850 (1999)：首款设备，双向寻呼机形态 + 拇指键盘 + 滚轮
- BlackBerry 5810 (2002)：首款增加电话功能
- 核心武器：Push Email + BES，端到端加密，占领华尔街/政府/律所
- 来源：src_383667, src_3454e2, src_ec6673

## 5. Palm OS / Treo 系列
- Palm Computing 1992 年创立，后 Jeff Hawkins 等另立 Handspring
- Treo 180 (2002)：首款 Palm OS 手机
- Treo 600 (2003)：QWERTY + 触屏 + 摄像头，最后一款 Handspring 品牌
- Treo 650 (2004)：Palm OS 5.4，最受欢迎全功能智能手机之一
- 来源：src_c9cc23, src_11ca9a, src_cd191d, src_44f9ab

## 6. Windows Mobile / Pocket PC
- 基于 Windows CE，2000 年推出 Pocket PC
- HTC 为核心代工厂
- 标志性产品：Orange SPV、Compaq iPAQ、O2 XDA
- 来源：src_a15a3c, src_3ef01d, src_340b2b, src_4bc483

## 7. 2006 年市场格局 (ABI Research)
- 全球智能手机出货量：7,090 万部
- Nokia：56.4%（4,000 万部）
- Motorola：8.5%，第二
- OS：Symbian ~73% > Windows Mobile > BlackBerry OS > Palm OS > Linux
- 来源：src_96fb0d

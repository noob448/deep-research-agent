# 笔记1: 前iPhone时代智能手机（1992-2006）

## 关键定义演变
- "智能手机"概念源于 PDA + 蜂窝通信融合
- Ericsson 内部原型 GS88（代号 Penelope）最早创造 "smart-phone" 一词
- 2000年 Ericsson R380 成为首款标称"Smartphone"的商用设备
- 成熟定义：开放操作系统 + 可安装第三方应用 + 互联网连接 + 邮件/消息处理

## 关键设备时间线

### IBM Simon (1994)
- 发明者: IBM 工程师 Frank Canova，1992年Comdex展出原型
- 1994年8月由BellSouth发售，$899（合约价）
- 功能: 电话+传真+电子邮件+寻呼机，触屏（需触笔），PCMCIA扩展
- 仅售出约5万部，6个月后停产
- 被广泛追认为"第一部智能手机"
- 来源: src_0a6f13 (Mobile Phone Museum), src_4b1a0e (Android Authority)

### Nokia 9000 Communicator (1996)
- 首款引入 QWERTY 物理键盘的手机，翻盖设计
- 8MB 内存、33MHz 处理器、运行 GEOS 操作系统
- 定义了"商务手机"品类
- 来源: src_5f5a31 (Medium), src_d9e7bf (Economic Times)

### Nokia 9210 Communicator (2001)
- 首批搭载 Symbian OS 的设备之一
- 彩色屏幕、多任务支持
- 来源: src_6fe6a2

### Ericsson R380 (2000)
- 全球第一款商业标称"Smartphone"的设备
- 翻起数字键盘暴露全尺寸触屏（360×120像素）
- 运行 Symbian EPOC Release 5
- 来源: src_0a6f13, src_ea7e88 (necrenie.com), src_ad8681

### BlackBerry (1999-2006)
- RIM 由 Mike Lazaridis 和 Jim Balsillie 于1984年创立
- RIM 850/857 Pager (1999) 为最初形态
- 2002年 BlackBerry 5810 首发 Push Email
- BES (BlackBerry Enterprise Server) 实现端到端加密和企业管理
- "CrackBerry" 一词形容用户依赖
- 来源: src_120a41 (Sprintzeal), src_e1ecad

### Palm Treo 系列 (2002-2006)
- Handspring 2002年推出首款 Treo (180/270/300)
- 2003年 Palm 收购 Handspring
- Treo 600 (2003): 首款真正消费化 Treo
- Treo 650 (2004): 最成功型号，312MHz XScale、320×320屏幕
- Treo 700w (2006): 转向 Windows Mobile
- 来源: src_08a8d1 (Low End Mac), src_f74be9

### Windows Mobile / Pocket PC (2000-2006)
- 基于 Windows CE，2000年发布 Pocket PC 品牌
- 2003年更名为 Windows Mobile
- WM 5.0 (2005) 引入持久化存储、GPS支持
- 关键 OEM: HTC、Compaq/HP、Dell
- 来源: src_86f792, src_9d7b1a (Computerhope), src_a15a3c (Wikipedia)

### Symbian OS
- 起源于 Psion EPOC (1994年开发32位EPOC32)
- 1998年 Symbian Ltd. 成立（Psion + Nokia + Ericsson + Motorola 合资）
- 分支: S60 (Nokia主导键盘)、UIQ (索爱触屏)、S80 (Communicator)、S90 (仅7710)
- 2006年达全球73%市场份额
- 来源: src_a85951 (Symbian Foundation), src_cd35d4 (Hotbot), [KB]

## 市场份额
- 2006年全球智能手机出货仅7,090万部

## 未覆盖/薄弱点
- Nokia 9300/9500 (2004) 细节偏少
- BlackBerry 具体型号发布日期不够精确
- HTC 等 Windows Mobile OEM 详细设备列表缺失
- Ericsson GS88 原型具体细节不足

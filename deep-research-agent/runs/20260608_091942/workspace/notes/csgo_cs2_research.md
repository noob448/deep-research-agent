# CS:GO Late Operations & CS2 Transition - Research Notes

## 1. Player Growth (2020-2023)

### Milestones:
- **February 11, 2023**: CS:GO broke all-time peak: **1,320,219 concurrent players** (Steam Charts) [src_7d753a]
- **April 2020**: Previous peak of **1,305,714 concurrent** during COVID-19 pandemic [src_7d753a]
- **March 2020**: CS:GO surpassed **1 million concurrent** for the first time [src_999479]
- **2022**: Hit 1,013,237 concurrent, recovering from 2021 dips [src_7d753a]
- **Late 2023 (CS2 release)**: Record-breaking concurrent user counts on Steam [src_353cc9]
- Average players Feb 2020: ~580k (55% increase YoY) [src_0b369c]

### Key Context:
- Free-to-play transition (2018) + COVID-19 pandemic created massive uptrend
- Game peaked during lockdowns when esports and online gaming surged

## 2. Operation Shattered Web (Nov 18, 2019 - Mar 30, 2020)

Source: [src_d05f0a] [src_aca8c2] [src_c06708] [src_5f2d63]

Key innovations:
- **First operation to introduce CT Operators and T Agents** — equippable character skins on any map
- **Battle Pass format** — replacing previous operation coin system
- New weapon collections, stickers, graffiti tags, crates with weapon skins
- SSG 553 rifle nerf (T-side)
- Weekly missions system with progress rewards
- Ninth CS:GO operation, succeeding Operation Hydra

## 3. Operation Broken Fang (2020-2021)

Source: [src_0e6d90] [src_796489]

Key features:
- **Tenth CS:GO operation**
- Continued battle pass format from Shattered Web
- Weekly missions, cosmetics, stat tracking system
- New agents and weapon collections

## 4. CS2 Announcement (March 22, 2023)

Source: [src_0dee93] [src_353cc9]

- **March 22, 2023**: Valve announced Counter-Strike 2
- Described as "largest technical leap" for the franchise
- **Free upgrade** for CS:GO owners
- Limited Test began same day (invite-only)
- Targeted "Summer 2023" full release
- CS:GO skins/inventory would carry over

## 5. CS2 Release (September 27, 2023)

Source: [src_353cc9] [src_0dee93]

- Released September 27, 2023, replacing CS:GO on Steam
- Record-breaking player counts on release day

## 6. CS2 Technical Features

### Sub-tick System
Source: [src_0488e4] [src_21ea84]

- Replaced fixed 64/128 tick model with sub-tick architecture
- Records exact timestamp of every input (mouse clicks, movement, shots) between ticks
- Server processes at 64Hz officially, but input timing is precise
- Eliminates 64-tick vs 128-tick debate
- Community feedback: mixed - better than 64-tick but some pros still prefer 128-tick feel

### Volumetric Smoke Grenades
Source: [src_c24e5f] [src_5fa781]

- Dynamic volumetric objects interacting with environment
- React to lighting, gunfire, and explosions
- HE grenades can temporarily clear smoke clouds
- Bullets create temporary sight lines through smoke

### Source 2 Engine
- Full engine migration from Source to Source 2
- Improved rendering, lighting, and physics
